# Cleverside — growth dashboard prototype

HTML/CSS prototype of a growth dashboard for funnel/experiment analysis (Spend, CPA, ROAS, conversion, retention/churn, revenue, audience segments).

- `index.html` — clickable prototype. Uses made-up example data (fictional funnel/experiment names and numbers) to illustrate structure and interaction — not real metrics.
- `dashboard-spec.md` — text spec: what each section shows, filter hierarchy, metrics glossary (what each metric means / how it's calculated).

Open `index.html` directly in a browser, or enable GitHub Pages for this repo to view it online.
